# css 작성방법 
## css와 hmtl 연결 방법 
    <link rel="stylesheet" href="./style.css"> : <head> 전에 작성하여 css와 html를 연결 시킨다 
## css를 통한 웹사이트 꾸미기 
### 색상 바꾸기 
    h1{
        color:색상;
        } :  h1을 "색상"의 컬러로 바꾼다 
       h1 말고 딴 html 요소의 색상 변경을 원할 시  = h1 대신 바꾸고 싶은 내용을 고르면 됨  
## div 활용법 
    <div>코딩 내용</div> : div 안에 있는 코딩 내용을 div 전체로 인식 
    * div{ color: 색상;} : div 안에 코딩 내용들이 "색상"의 컬러로 바뀜
